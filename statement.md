PROBLEM STATEMENT :

The core problem can be broken down into three main points concerning the inaccessibility and inefficiency of traditional banking and the need for modern developer training

The system allows user to :
1. Login to the atm account using pin
2. View a 5 numbered options
3. Check the balance
4. Deposit the amount
5. Withdraw the amount

SCOPE OF THE PROJECT:

The scope of an ATM Interface System project developed in Python primarily focuses on simulating the core functionalities of a traditional Automated Teller Machine in a secure, text-based environment.

TARGET USERS:

The project targets developers (for training in secure banking logic) and general users (for practicing basic, user-friendly financial transactions).
